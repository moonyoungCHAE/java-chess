# java-chess
체스 게임 구현을 위한 저장소

## 요구사항
* 체스판을 초기화한다.
    * 체스판에서 말의 위치 값은 가로 위치는 왼쪽부터 a ~ h이고, 세로는 아래부터 위로 1 ~ 8로 구현한다.
    * 체스판에서 각 진영은 검은색(대문자)과 흰색(소문자) 편으로 구분한다.
        * 검은색은 세로 7, 8줄에 위치한다.
        * 흰색은 세로 1, 2줄에 위치한다.
* 게임 명령어 도움말을 출력한다.
* 게임 명령어를 입력 받는다.
    * start: 게임 시작
    * end: 게임 종료
    * (예외) 게임을 시작하지 않았는데, 종료를 입력했을 때
    * (예외) 명령어를 정확하지 입력하지 않았을 때
